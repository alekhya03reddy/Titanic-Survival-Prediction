# Titanic Survival Prediction

This project predicts whether a passenger survived the Titanic disaster using Machine Learning techniques.

## Objective
To build a machine learning model that predicts passenger survival based on features like age, gender, passenger class, fare, and embarked location.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Algorithm
- Logistic Regression

## Features Used
- Passenger Class
- Gender
- Age
- Fare
- Family Members
- Embarked

## Accuracy
Approximately 80%

## Dataset
Titanic Dataset from Kaggle

## Conclusion
Successfully built a Titanic Survival Prediction model using Logistic Regression with good prediction accuracy.
